pIRC - The peer to peer IRC client!
==================================
Authors: 5a1t, rtsio


Setup:
	+src
	|----->chat-client (node IRC client functions)
	|----->chat-server (node IRC server functions)
	|----->networking (p2p networking)
	|----->ui (ui for user input)
	+lib
	+pIRC.py (main program)


Reference:
	-https://tools.ietf.org/html/rfc2812 (IRC Protocol)
